# SSC – ShocKing Service Calculator

פרויקט React + Vite + Tailwind + Framer Motion שמכיל את מחשבון טיפולי הבולמים שלך.

## הרצה מקומית

```bash
npm install
npm run dev
```

העמוד ירוץ כברירת מחדל ב- http://localhost:5173/

## בנייה להפקה

```bash
npm run build
npm run preview
```

---

הקובץ המרכזי נמצא ב- `src/App.tsx`.
